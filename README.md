# Bokeh_Assignment
One of the main advantages of using Bokeh is its ability to create interactive visualizations. With Bokeh, you can create a wide range of interactive elements.This interactivity can be particularly useful in data dashboards, where users need to be able to quickly explore and understand complex data sets.
